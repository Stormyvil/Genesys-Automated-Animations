# Genesys Automated Animations

This is a small companion module for Foundry VTT that makes the Automated Animations module work with the Genesys system.

Automated Animations doesn't recognise Genesys attack rolls. So even if you've configured animations on your weapons, nothing plays. This module sits in the background and bridges the two, automatically triggering animations and sounds when a character makes an attack roll.

Configure animations on your weapons the same way you normally would in Automated Animations 


## Requirements
- [**Automated Animations**](https://foundryvtt.com/packages/autoanimations)
- [**Sequencer**](https://foundryvtt.com/packages/sequencer) (required by Automated Animations)

## Recommended
- [**JB2A**](https://foundryvtt.com/packages/JB2A_DnD5e/)

For more info about Automated Animations functionality, see the author [**here**](https://github.com/theripper93/autoanimations?tab=readme-ov-file)
